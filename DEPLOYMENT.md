# Deployment Guide

## Vercel Deployment

1. Go to [vercel.com](https://vercel.com)
2. Click "New Project"
3. Import your GitHub repository
4. Add environment variables:
   - REACT_APP_SUPABASE_URL
   - REACT_APP_SUPABASE_ANON_KEY
   - REACT_APP_OPENAI_API_KEY
   - REACT_APP_RESEND_API_KEY
